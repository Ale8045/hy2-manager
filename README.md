# HK HY2 Manager

一个支持国内/国外 VPS、多节点、IP/SSL 双模式的一键 HY2 管理脚本。

---

# 🚀 一键安装

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Ale8045/hy2-manager/main/install.sh)
```

安装完成后输入：

```bash
hy2
```

即可打开管理面板。

---

# 🔐 SSL功能

支持：

- 自动申请 SSL
- 自动续期
- 查看证书状态
- 查看剩余有效期
- 自动切换 SSL 模式

推荐使用 Cloudflare 泛解析。

---

# 🌐 支持模式

## IP模式

适合快速部署：

```
insecure=1
```

---

## SSL模式

正式证书：

```
insecure=false
```

更加稳定。

---

# ⚠ 注意事项

## 云服务器安全组

必须放行对应节点 UDP 端口。

例如：

```
UDP 48357
```

---

## SSL模式

域名需解析到服务器 IP。

支持：

- Cloudflare
- 泛域名解析
